# 背景粒子特效

10种 Minecraft 风格的动态背景特效 ✨

</div>
---

## 简介

背景粒子特效是一款 SJMCL 扩展，提供 10 种 Minecraft 风格的动态粒子背景特效，可让你的启动器主页更生动。

可与 sjmcl-city-weather（城市天气粒子）、sjmcl-video-background（动态视频背景）叠加使用，打造个性化启动器界面。

> 📌 同时开启视频背景时，若要切换粒子特效需先在扩展卡片删除视频再调整。

## 特效列表

| 特效 | 说明 |
|------|------|
| ☀️ 晴天 | 明亮温暖的阳光粒子 |
| 🌧 雨 | 淅淅沥沥的雨滴效果 |
| ❄️ 雪 | 飘落的雪花粒子 |
| 🌪 沙暴 | 漫天黄沙的沙漠风暴 |
| ✨ 星空 | 闪烁的夜空星群 |
| 🌌 极光 | 流动的极光帷幕 |
| 🍂 落叶 | 随风飘舞的秋叶 |
| 🌊 水下 | 水中气泡与光线折射 |
| 🔥 火焰 | 跳动的烈焰粒子 |
| 🌅 日落 | 暖色调的落日余晖 |
| ⛰ MC风景 | Minecraft 风格的风景粒子 |

## 使用方法

1. 在 Releases 下载 org.sjmcl.weather-effects.sjmclx
2. 打开 SJMCL → 设置 → 扩展管理 → 导入，插件支持选择文件导入和拖动导入
3. 在扩展设置页或首页卡片中选择喜欢的特效
